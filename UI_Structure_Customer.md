### Component Breakdown

- **Header**
  - Left: Lucide `MapPin` + current location name (truncated), tap to change
    - Dropdown/bottom sheet with saved addresses + map picker
  - Right: Lucide `Bell` icon + notification badge (red dot with count)
- **Greeting**
  - "Good morning/afternoon/evening, [First Name]! рҹ‘Ӣ"
  - Font: Montserrat Bold, 20px, `#0F172A`
  - Time-aware greeting logic
- **Book CTA Card**
  - Full width card, `#DBEAFE` background tint
  - "What do you need done?" вҖ” 16px, `#0F172A`
  - "Book an Errand вҶ’" вҖ” 16px, `#2563EB`, bold
  - Tap вҶ’ navigates to Booking Flow Step 1
- **Quick Actions Grid** (errand type shortcuts)
  - Layout: 4 columns row 1, 3 columns row 2
  - Each item: 72dp box, icon (Lucide) on `#F8FAFC` circle, label below
  - Icons: `Package` (Delivery), `ShoppingCart` (Purchase), `Banknote` (Bills), `FileText` (Docs), `Clock` (Queue), `Car` (Ride), `Settings` (Custom)
  - Tap вҶ’ jumps to Booking Flow Step 2 with type pre-selected
- **Active Errand Card** (conditional вҖ” hidden if no active errand)
  - Status indicator dot: green `#22C55E`
  - Status text, errand type, runner name
  - ETA in bold
  - "Track вҶ’" link in `#2563EB`
  - Tap card вҶ’ navigates to Live Tracking screen
- **Recent Errands List**
  - Component: `Card` per errand
  - Shows: date, errand type, amount in вӮұ, star rating, runner name, status
  - "Re-book" button: ghost/outline, `#2563EB` text
  - Max 3 shown, "View All вҶ’" link to Activity tab
- **Promos Banner** (conditional вҖ” shown if active promos exist)
  - Horizontal scrollable cards
  - Background: `#DBEAFE`, promo text, expiry date
  - Tap вҶ’ auto-fills promo code in booking flow

- **Schedule Options** (radio cards)

  - "Schedule for later": Lucide `Calendar` icon
  - Same selectable card styling as Step 1
- **Calendar** (conditional вҖ” shown when "Schedule" selected)
  - Component: Custom calendar picker
  - Today highlighted with blue dot
  - Selected date: `#2563EB` filled circle, white text
  - Past dates: grayed out, disabled
  - Max 7 days from today
  - Swipe left/right for month navigation
- **Time Picker**
  - Scroll-wheel style or segmented hour:minute picker
  - AM/PM dropdown
  - Minimum time: if today, must be at least 1 hour from now
- **Next Button** вҖ” proceeds to Step 4
